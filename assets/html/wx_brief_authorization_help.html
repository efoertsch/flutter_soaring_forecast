Aeronautical and meteorological information is obtained from 1800wxbrief.com. So before using this option you must first authorize 1800wxbrief to allow this app to request flight information from it. To do this:

1. Go to 1800WxBrief.com
2. Click on "GO TO DESKTOP SITE", if the mobile version of the site is displayed.
3. Sign on to 1800WxBrief.
4. On the menu bar click on Account/Service Provider Authorization
5. Click Edit
6. Click Yes next to SoaringForecast
7. Click Save

Doing this will not hinder your access to 1800WxBrief. You can continue to use 1800WxBrief as before.
